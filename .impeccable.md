# Impeccable Design Context

## Design Context

### Users
- **主要受众**: 国际教育行业从业者（合作伙伴、主管、业务负责人），关注 AI 如何提升工作效率
- **次要受众**: AI 实践爱好者、个人品牌关注者，通过博客/Newsletter 发现网站
- **使用场景**: 阅读博客文章、了解 AI 实训服务、建立业务联系
- **核心任务**: 快速判断"这个人值得关注/合作"，然后订阅或联系

### Brand Personality
- **三词定义**: 创新 · 果断 · 前沿
- **语调**: 自信但不傲慢，数据驱动但有温度，技术前沿但接地气
- **情绪目标**: 信任感（靠谱实干）+ 启发感（AI 新视角）+ 专业敬畏（行业领袖）+ 亲近感（真实可交流）

### Aesthetic Direction
- **视觉基调**: 现代科技感 + 内容优先的简洁排版，不追求花哨但要有视觉冲击力
- **不想要**: 过于朴素/学术风；避免纯白无色彩的极简主义
- **当前方向**: 暖白背景 + 深蓝强调 + 琥珀 CTA，方向正确但需要更大胆的视觉表现
- **主题**: 同时支持 Light 和 Dark mode
- **图标体系**: Lucide React

### Typography
- **西文**: Inter (variable)
- **中文**: Noto Sans SC, PingFang SC, Hiragino Sans GB fallback
- **Prose**: 1.0625rem / line-height 1.8
- **代码**: SF Mono, Fira Code

### Design Principles

1. **Content is King, Design is Amplifier** — 内容是核心资产，设计的目的是让内容更有说服力，不是分散注意力。每个视觉元素都应该服务于阅读体验或转化目标。

2. **Confident, Not Loud** — 用克制的方式传达力量感。大胆的排版层级、精准的色彩点缀、充足的留白，而不是渐变、动画堆叠。果断的设计决策本身就是自信的表达。

3. **Data-Grounded Trust** — 数字和事实是信任的基石。重要指标（3万+合作伙伴、40万客户、75亿交易）需要突出但不浮夸的呈现方式。设计要让数据自然醒目。

4. **Dual-Theme Ready** — 所有颜色通过 CSS 变量定义，组件不硬编码色值。Dark mode 不是 Light mode 的反转，而是独立调优的视觉体验。

5. **Chinese-First, Bilingual-Ready** — 排版以中文阅读体验为主优化（行高、字间距、段落间距），西文作为辅助。中文 font-weight 600 = 视觉粗体，避免 700+ 造成中文笔画粘连。
