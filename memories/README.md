# Memories 目录

将各个 Claude Code 开发项目的 CLAUDE.md 文件放入此目录。

## 文件命名规范

按项目命名，例如：

```
memories/
├── uhomes-talent.md        # 异乡人才项目
├── uhomes-pay.md           # 异乡缴费项目
├── uhomes-review.md        # 异乡点评项目
├── dingning-ai.md          # 个人网站项目
├── ai-news-bot.md          # AI 新闻机器人
└── ...
```

## 文件内容格式

直接复制 Claude Code 的 CLAUDE.md 内容即可。脚本会自动从中提取项目信息、技术栈、关键决策等内容作为文章素材。

## 更新频率

建议每周同步一次各项目的最新 CLAUDE.md 到此目录。
