# dingning.ai 个人网站

## 项目概述
个人品牌网站，定位为国际教育 AI 布道者。使用 Next.js 14 + MDX 博客系统。

## 项目时间线
- 2026-03-07：GitHub 仓库创建，网站上线
- 2026-03-07 至今：持续迭代（博客系统、自动生成、CI/CD）

## 技术栈
- Next.js 14 (React 18) + TypeScript
- Tailwind CSS
- MDX 内容管理 (gray-matter + next-mdx-remote)
- Vercel 部署
- Buttondown Newsletter 订阅

## 关键决策
- 选择 MDX 而非 CMS，保持内容版本化和本地可控
- 使用 CSS 变量实现主题系统，暖白底色 + 海军蓝强调色
- 导航和 UI 文案全部使用中文
- 博客文件命名采用 YYYY-MM-DD-slug.mdx 格式

## 开发历程
- Phase 1：首页 + 全局布局 + 5 页面骨架
- Phase 2：MDX 博客系统 + Newsletter + 7 篇原创文章
- 后续迭代：品牌一致性、资源页重构、服务页面、AI 布道者定位

## 经验教训
- Vibe Coding 适合快速搭建内容型网站
- 非程序员完全可以用 AI 辅助完成生产级网站开发
- Git 版本管理 + Code Quality 门禁是必须的工程实践
