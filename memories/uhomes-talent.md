# 异乡人才 (UHomes Talent)

## 项目概述
AI 驱动的留学生求职平台，帮助留学生找到回国就业机会。目前有 10,645 个活跃岗位。

## 项目时间线
- 2026-02-13：GitHub 仓库创建，项目启动
- 2026-02 至 2026-03：核心功能开发（推荐引擎、岗位抓取、前端）
- 2026-03：上线运营，持续迭代中

## 技术栈
- Python + Django 后端
- React 前端
- PostgreSQL 数据库
- Redis 缓存
- 使用 Thompson Sampling 的智能推荐引擎

## 关键功能
- 岗位智能匹配：基于学历、专业、语言能力等维度
- 顾问评价系统：505 个单元测试保障的 Thompson Sampling 排名算法
- 自动化岗位抓取：每日自动更新岗位数据

## 开发亮点
- 全程 Vibe Coding，非程序员独立完成 18,000+ 行代码
- 505 个单元测试确保推荐算法可靠性
- 采用 Gate-Review 评审方法论控制产品质量

## 经验教训
- Thompson Sampling 比简单评分平均值更公平
- 单元测试是 Vibe Coding 的安全网
- 非程序员也能理解和应用复杂算法，关键是把"为什么"讲清楚
