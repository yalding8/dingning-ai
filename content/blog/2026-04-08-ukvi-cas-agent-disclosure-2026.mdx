> **本文读者定位**：以 B 端为主——招生机构、master agent、sub-agent、英国院校国际办公室、留学行业从业者。C 端学生家长可参考第一部分了解概念边界。

最近朋友圈在传一条消息：

> ❗英国签证与移民局（UKVI）就录取确认函（CAS）签发事宜发布了一项重要更新❗
> 自 2026 年 4 月 7 日起，所有院校在向学生签发 CAS 时，会在 CAS 中体现中介机构信息（中介名称、联系人及中介地址）。
> 此属官方新规要求，不影响签证及入学。

这条消息**核心属实**，但有几处措辞需要修正——尤其是"中介"两个字背后藏着一个非常重要的概念区分。"不影响签证及入学"这句话也只说了表层。

我花了一些时间核对 UK Government 一手文件，整理出这份事实清单。本文重点不在"给学生的建议"，而在**对 B 端的影响分析**——招生机构、master agent、sub-agent、英国院校如何被这条政策重塑。

## 第一部分：先把"中介"这个词说清楚

这条新规里说的"中介"，**不是**多数中国留学生熟悉的那种留学顾问。它有一个非常具体的英文定义：**education agent / recruitment agent**——也就是**与英国院校签有招生合作合同的代理招生机构**。

这两件事常常被混为一谈，但它们是不同的概念：

| 概念 | 英文 | 含义 | 是否进入 CAS |
|---|---|---|---|
| **代理招生机构** | recruitment agent / education agent | 与英国院校签有正式招生合同的机构，被授权代表院校接收申请 | ✅ 是 |
| **留学顾问 / 留学服务机构** | student advisory / education consultancy | 学生付钱购买服务的消费者端机构，提供咨询、文书、签证辅导 | ⚠️ 仅在它本身是 agent 或上游有 agent 关系时 |

CAS 上的"agent"特指**和院校签有合同的那一方**。如果某留学顾问通过 sub-agent 模式挂在某个 master agent 下面，CAS 上填的是上层 master agent 的名字——和学生交钱的对象不一定是同一家。

**事实依据**：[British Council — Good Practice Guide for UK Providers Using Education Agents (PDF)](https://www.britishcouncil.org/sites/default/files/good-practice-guide-for-uk-providers-using-education-agents.pdf) 明确将 agent 定义为 "contracted directly or indirectly to deliver a range of services"，并专章说明 sub-agent / master agent 的法律关系。

这个概念边界是理解后续分析的前提。**新规的真正主体是 B 端的代理招生链条，不是消费者端的留学顾问市场**——尽管两者高度交叉。

## 第二部分：事实核查

### 哪些是真的，哪些需要修正

| 主张 | 状态 | 依据 |
|---|---|---|
| 2026 年 4 月 7 日生效 | ✅ 事实 | Student Sponsor Guidance 文件 1 & 2 在该日期更新生效 |
| CAS "强制" 显示中介信息 | ⚠️ 措辞需修正 | 准确说法：CAS 表单有勾选框"是否通过 agent 招生"。**勾选后**，agent 主要联系人姓名等成为必填项 |
| "所有院校所有 CAS" | ⚠️ 不准确 | 仅适用于"通过 agent 招生"的情况；DIY 申请人不受影响 |
| 不影响签证及入学 | ⚠️ 半真 | 单笔申请短期不受影响。中长期机制见下文 |

**事实依据**：UK Government [SMS Guide 4A — Creating a CAS for Student Sponsors](https://www.gov.uk/government/publications/certificates-of-sponsorship-cos-sms-user-manual/sms-guide-4a-creating-a-cas-guide-for-student-sponsors-accessible) 明确写道：申请表中的 agent details 字段包含一个勾选框"是否通过 agent 招收该学生"，勾选后必须填写 agent 主要联系人姓名等字段。

### 政策时间线（事实部分）

- **2022 年**：UK Agent Quality Framework（AQF）经过多年研究后建立，由 British Council、BUILA、UUKi、UKCISA 联合主导。**注意：AQF 是行业自律框架，不是 UKVI 强制监管**
- **2023 年 12 月**：AQF 正式启动 [British Council 一手页面](https://www.britishcouncil.org/education/agents-counsellors/uk-quality-agent-framework)
- **2024 年**：Migration Advisory Committee（MAC）发布国际学生政策审查，建议加强 agent 透明度
- **2025 年**：UKVI 宣布将开发系统能力，把 agent 信息加入 CAS——预计 2025 年夏季技术就绪
- **2026 年 1 月**：Sponsor Management System（SMS）字段规范更新
- **2026 年 4 月 7 日**：更新后的 Student Sponsor Guidance 文件 1 & 2 正式生效（同日远程授课规则也生效）
- **2026 年 6 月**：新版 Basic Compliance Assessment（BCA）框架启用

**事实依据**：[GOV.UK Student Sponsor Guidance](https://www.gov.uk/government/publications/student-sponsor-guidance)、[UKCISA Student Sponsor Licence Changes](https://www.ukcisa.org.uk/student-advice/visas-and-immigration/student-sponsor-licence-changes/)、[The PIE News: UK agent details to be added to CAS](https://thepienews.com/uk-agent-details-to-be-added-to-cas-by-summer-2025/)。

### 新版 BCA 评估门槛（事实）

2026 年 6 月起，每所英国大学需满足：

- **签证拒签率 < 5%**
- **学生注册率 ≥ 95%**
- **课程完成率 ≥ 85%**

任何一项不达标，院校的 student sponsor licence 会进入风险状态。

**事实依据**：[GOV.UK Student Sponsor Compliance](https://www.gov.uk/government/publications/student-sponsor-guidance/student-sponsor-compliance-accessible)。

## 第三部分：B 端影响分析

下面进入分析部分。每段标注依据强度。

### 一、对英国院校国际办公室

#### [推论 · 依据强度：中] 院校 BCA 压力会传导成 agent 管理压力

事实层面：BCA 5% 拒签率门槛是硬性的，不达标直接影响 sponsor licence。CAS 上现在有了 agent 字段——意味着院校自身可以**按 agent 维度反查**自己签发 CAS 的拒签分布。

合理推论：每一所英国大学的国际办公室会增加一项工作——**定期复盘各 agent 的拒签率和注册率**，并据此调整合作名单。这是 BCA 框架下最直接的 self-defense 动作。

依据：BCA 门槛（事实）+ CAS agent 字段（事实）+ 院校 sponsor licence 经济激励（事实）。但"院校建立 agent KPI 看板"是推断，**不是 UKVI 强制要求**。

#### [推论 · 依据强度：强] 罗素集团 vs 二三线院校的合规能力差距会被放大

事实层面：罗素集团院校普遍有专职 international compliance team，二三线院校的国际办公室通常人手紧张。新规带来的 agent 管理工作量是真实的、必须完成的。

合理推论：能管好 agent 网络的院校继续招生，管不好的会出现 BCA 风险——进而**主动收缩 agent 数量**或退出某些高风险市场。中长期看，这会重塑英国高教的国际化版图。

依据：院校合规人力配比差异（行业普遍现象）+ BCA 不达标后果（事实）。

#### [推论 · 依据强度：弱] 院校可能开始建立内部 agent 黑名单

如果 BCA 压力持续传导，院校理性的反应是建立内部的 agent 评级或黑名单。这件事不会公开，但会通过"我们已经停止接受某 agent 的申请"这种邮件回复体现出来。

依据：纯粹基于经济激励的逻辑推演。**没有公开案例支撑**。

### 二、对 master agent

#### [推论 · 依据强度：强] master agent 进入"代人受过"的位置

事实层面：sub-agent 模式下，CAS 上填的是 master agent 的名字，不是底层 sub-agent。这是 sub-agent 合同结构的常态做法。

合理推论：当某个 sub-agent 的学生出现高拒签率时，**承担数据后果的是 master agent**——名字出现在 CAS 上、被院校反查、被列入观察名单的也是 master agent。这意味着 master agent 必须重构对 sub-agent 的尽调和质量管控机制。过去松散的 "挂靠 + 抽佣" 模式不再可持续。

依据：sub-agent 法律结构（事实）+ CAS 字段填的是合同方（事实）。结论"master agent 必须重构尽调"是商业判断，不是政策要求。

#### [推论 · 依据强度：中] master agent 的护城河可能被强化

如果院校开始基于 CAS 数据筛选 agent，**已经建立合规体系、有清晰 sub-agent 管理流程的 master agent 会成为院校的首选**。新增 agent 进入院校合作名单的门槛会提高。

依据：BCA 压力 → 院校筛选 agent（中等强度推论） + 现有 master agent 的运营壁垒（行业常识）。

### 三、对 sub-agent / 二三线留学机构

#### [推论 · 依据强度：中] sub-agent 面临 "被上游清退" 的风险

事实层面：master agent 现在为每一个 sub-agent 的学生数据负责。

合理推论：master agent 会主动评估自己的 sub-agent 网络——质量差的 sub-agent 会被解约，因为继续合作意味着把 master agent 自己的合规分数拖下来。**这是新规对二三线市场最直接的冲击**：sub-agent 不再只是和学生博弈，也要和上游 master agent 博弈。

依据：上一节关于 master agent 的推论。

#### [推论 · 依据强度：弱] 部分 sub-agent 可能转向"无 agent 关系" 的灰色操作

如果被上游清退后无法找到新的 master agent 挂靠，部分 sub-agent 可能会转向"指导学生 DIY 申请、不勾选 agent 字段"的灰色操作——表面看是 DIY，实际仍在收取顾问费。

依据：行为推测，无公开案例。这种操作的合规风险很高，但市场上确实存在动机。

### 四、对头部直营招生机构（既是 agent 又是 to-C 服务方）

#### [推论 · 依据强度：强] 头部直营机构是新规的最大受益者

这里的"头部直营"指的是**既和英国院校直接签有 agent 合同、又直接服务 C 端学生**的机构。这种双重身份在新规下有结构性优势：

- 他们的 sub-agent 链条短或没有 sub-agent，数据归属清晰
- 他们有专业合规团队，可以快速适配 SMS 字段更新
- 他们在 CAS 上出现的就是自己的名字，品牌可见度上升
- 院校筛 agent 时，他们是天然的"低风险选项"

依据：业务结构差异（事实）+ 合规适配成本（事实）+ 院校筛选逻辑（中等强度推论）。

### 五、对中国市场的整体影响

#### [推论 · 依据强度：中] 这是中国 B 端市场的一次"代理链条透明化"

绝大多数中国学生通过留学顾问 / agent 链条申请，这一比例显著高于英国其他主要生源国。这意味着新规对中国 B 端市场的实际影响倍数最高。

但更有意思的观察是：**中国 B 端市场存在大量 sub-agent / 灰色挂靠关系**。新规打开的不只是"哪家 agent 服务了哪个学生"这个问题，而是把整条挂靠链暴露在院校面前。这是中国市场过去十几年第一次面对这种透明度。

依据：中国市场 sub-agent 渗透率（行业共识，无精确数据）+ CAS 字段填合同方（事实）。

## 第四部分：给申请人的最低限度提示

本文重点是 B 端，但对 C 端申请人有一句最重要的话：

**你交钱的留学顾问、与你的英国院校签合同的 agent、最终出现在你 CAS 上的机构，可能是三家不同的公司。**

你至少应该知道这三家是谁。这是新规给申请人带来的**唯一真正可操作的信息**——在签合同前问清楚你的留学顾问在 agent 链条里的位置。

## 结论

**事实部分**：

- 2026/4/7 起，CAS 表单中"通过 agent 招生"勾选后必须填写 agent 信息
- 这里的 agent 是与院校签有合同的代理招生机构，**不一定是**学生付钱的留学顾问本身
- 2026/6 起新 BCA 框架要求拒签率 < 5%
- agent 信息从此公开可追溯

**B 端推论部分**：

- **院校层面**：BCA 压力传导成 agent 管理压力；罗素集团 vs 二三线院校的合规能力差距被放大
- **master agent 层面**：进入"代人受过"位置，必须重构 sub-agent 尽调机制；同时其护城河被强化
- **sub-agent 层面**：面临被上游清退的风险；部分可能转向灰色操作
- **头部直营机构**：新规的结构性受益者
- **中国市场整体**：第一次面对代理链条的透明化

这些**不是确定的未来**，是基于已知事实的合理推演——读者应当知道二者的区别。

**对 B 端从业者的核心判断**：

新规的真正影响不在"信息公开"本身，而在**信息公开之后院校的反应**。院校 BCA 不达标的代价是 sponsor licence——这是英国大学的核心生命线。一旦院校开始把 BCA 风险传导到 agent 选择上，整个代理链条会经历一次结构性洗牌。

**头部机构受益、master agent 角色升级、sub-agent 重新洗牌、灰色挂靠空间收窄**——这是这条新规真正的影响版图。
