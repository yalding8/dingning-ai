最近朋友圈在传一条消息：

> ❗英国签证与移民局（UKVI）就录取确认函（CAS）签发事宜发布了一项重要更新❗
> 自 2026 年 4 月 7 日起，所有院校在向学生签发 CAS 时，会在 CAS 中体现中介机构信息（中介名称、联系人及中介地址）。
> 此属官方新规要求，不影响签证及入学。

这条消息**核心属实**，但有几处措辞需要修正——尤其是"中介"两个字背后藏着一个非常重要的概念区分。"不影响签证及入学"这句话也只说了表层。

我花了一些时间核对 UK Government 一手文件，整理出这份事实清单和影响分析。

## 第一部分：先把"中介"这个词说清楚

这条新规里说的"中介"，**不是**多数中国留学生熟悉的那种留学顾问。它有一个非常具体的英文定义：**education agent / recruitment agent**——也就是**与英国院校签有招生合作合同的代理招生机构**。

这两件事常常被混为一谈，但它们是不同的概念：

| 概念 | 英文 | 含义 | 是否进入 CAS |
|---|---|---|---|
| **代理招生机构** | recruitment agent / education agent | 与英国院校签有正式招生合同的机构，被授权代表院校接收申请 | ✅ 是 |
| **留学顾问 / 留学服务机构** | student advisory / education consultancy | 学生付钱购买服务的消费者端机构，提供咨询、文书、签证辅导 | ⚠️ 不一定 |

**关键区别**：

- 部分大型机构**同时**是两种身份——它既给学生提供服务，又和英国院校签有正式 agent 合同。这种情况下，CAS 上会出现这家机构的名字
- 但**很多留学顾问**只是帮学生包装材料、递交申请，**本身没有任何英国院校的招生合同**。这种情况下：
  - 如果该顾问通过 master agent / sub-agent 模式挂在某个有合同的大 agent 下面，CAS 上填的是上层 master agent 的名字
  - 如果该顾问完全没有任何上下游 agent 关系，理论上院校在签 CAS 时根本不应该勾选"使用了 agent 招生"这个选项
  - 实际操作中，部分留学顾问会以 sub-agent 身份挂靠，于是 CAS 上出现的是另一家机构的名字——和学生交钱的对象不是同一家

**这件事对申请人意味着什么**：你需要主动问你的留学顾问两个问题——

1. 你和我申请的英国院校是什么样的合作关系？是直接 agent，还是 sub-agent，还是没有任何官方关系？
2. 在我的 CAS 上会出现哪家机构的名字？

**事实依据**：[British Council — Good Practice Guide for UK Providers Using Education Agents (PDF)](https://www.britishcouncil.org/sites/default/files/good-practice-guide-for-uk-providers-using-education-agents.pdf) 明确将 agent 定义为"contracted directly or indirectly to deliver a range of services"，并专章说明 sub-agent / master agent 的法律关系。

## 第二部分：事实核查

### 哪些是真的，哪些需要修正

| 主张 | 状态 | 依据 |
|---|---|---|
| 2026 年 4 月 7 日生效 | ✅ 事实 | Student Sponsor Guidance 文件 1 & 2 在该日期更新生效 |
| CAS "强制" 显示中介信息 | ⚠️ 措辞需修正 | 准确说法：CAS 表单有勾选框"是否通过 agent 招生"。**勾选后**，agent 主要联系人姓名等成为必填项 |
| "所有院校所有 CAS" | ⚠️ 不准确 | 仅适用于"通过 agent 招生"的情况；DIY 申请人或未通过任何 agent 申请的不受影响 |
| 不影响签证及入学 | ⚠️ 半真 | 单笔申请短期不受影响。中长期机制见下文分析 |

**事实依据**：UK Government [SMS Guide 4A — Creating a CAS for Student Sponsors](https://www.gov.uk/government/publications/certificates-of-sponsorship-cos-sms-user-manual/sms-guide-4a-creating-a-cas-guide-for-student-sponsors-accessible) 明确写道：申请表中的 agent details 字段包含一个勾选框"是否通过 agent 招收该学生"，勾选后必须填写 agent 主要联系人姓名等字段。

### 政策时间线（事实部分）

- **2022 年**：UK Agent Quality Framework（AQF）经过多年研究后建立，由 British Council、BUILA、UUKi、UKCISA 联合主导。**注意：AQF 是行业自律框架，不是 UKVI 强制监管**
- **2023 年 12 月**：AQF 正式启动 [British Council 一手页面](https://www.britishcouncil.org/education/agents-counsellors/uk-quality-agent-framework)
- **2024 年**：Migration Advisory Committee（MAC）发布国际学生政策审查，建议加强 agent 透明度
- **2025 年**：UKVI 宣布将开发系统能力，把 agent 信息加入 CAS——预计 2025 年夏季技术就绪
- **2026 年 1 月**：Sponsor Management System（SMS）字段规范更新
- **2026 年 4 月 7 日**：更新后的 Student Sponsor Guidance 文件 1 & 2 正式生效（同日远程授课规则也生效）
- **2026 年 6 月**：新版 Basic Compliance Assessment（BCA）框架启用

**事实依据**：[GOV.UK Student Sponsor Guidance](https://www.gov.uk/government/publications/student-sponsor-guidance)、[UKCISA Student Sponsor Licence Changes](https://www.ukcisa.org.uk/student-advice/visas-and-immigration/student-sponsor-licence-changes/)、[The PIE News: UK agent details to be added to CAS](https://thepienews.com/uk-agent-details-to-be-added-to-cas-by-summer-2025/)。

### 新版 BCA 评估门槛（事实）

2026 年 6 月起，每所英国大学需满足：

- **签证拒签率 < 5%**
- **学生注册率 ≥ 95%**
- **课程完成率 ≥ 85%**

任何一项不达标，院校的 student sponsor licence 会进入风险状态。

**事实依据**：[GOV.UK Student Sponsor Compliance](https://www.gov.uk/government/publications/student-sponsor-guidance/student-sponsor-compliance-accessible)。

### CAS 撤回与重新签发（事实——更正之前版本）

这是一条需要专门澄清的事实——之前传言里有种说法是"CAS 一旦签发，agent 信息就锁定"，**这不准确**。

**真实规则**：

- 在学生**提交签证申请之前**，院校可以撤回（withdraw）已签发但未使用的 CAS
- 撤回之后，院校可以**重新签发**一份新的 CAS（理论上可以填写更新后的 agent 信息）
- 一旦学生**已经提交签证申请**，CAS 就和该次签证申请绑定，不能再更改

所以"换 agent"的事实边界是这样的：

- 签证申请提交**前**换 → 可操作（需要原院校配合撤回原 CAS、重发新 CAS）
- 签证申请提交**后**换 → 不可操作

**事实依据**：[UK GOV — Sponsorship Duties](https://www.gov.uk/government/publications/student-sponsor-guidance/sponsorship-duties-accessible) 关于 CAS 撤回的条款；[SMS Guide 4A](https://www.gov.uk/government/publications/certificates-of-sponsorship-cos-sms-user-manual/sms-guide-4a-creating-a-cas-guide-for-student-sponsors-accessible) 关于 CAS 创建与修改的规定。

## 第三部分：分析与推论

到这里为止，所有内容都是可核实的事实。下面进入分析部分——我会清晰标注每段的依据强度。

### [推论 · 依据强度：中] 拒签率开始按 agent 维度统计

UKVI 既然要求 CAS 上记录 agent 信息，**技术上**就具备按"招生 agent"维度聚合学生签证数据的能力。但这件事是否会实际发生——UKVI 是否会公布机构维度的拒签率数据、院校自身是否能访问这种粒度的数据——目前没有公开承诺。

**依据**：CAS 字段加入 agent 信息（事实）+ AQF 框架明确目标包含降低 visa-mill 链条风险（事实）。但"院校能看到 agent 维度数据"这一步**没有官方文件确认**。

### [推论 · 依据强度：中] 院校的 BCA 分数会被 agent 质量影响

如果某 agent 名下学生的拒签率持续偏高，并且这些数据进入 BCA 总体计算（这是 BCA 的逻辑），院校就有动机管控其合作的 agent。BCA 一旦不达标，影响的是整所大学未来几年的国际招生能力——这是英国大学的核心收入来源之一。

**依据**：BCA 5% 拒签率门槛（事实）+ sponsor licence 影响院校生存（事实）。但"院校据此筛选具体 agent"是合理推断，**不是 UKVI 公布的强制机制**。

### [推论 · 依据强度：弱] 院校层面的二次筛选

如果上述机制成立，院校最理性的反应不是事后申诉，而是事前筛选合作 agent。某些 agent 可能会发现"我们和某某大学的合作通道关闭了"。

**依据**：纯粹基于经济激励的逻辑推演。**没有公开案例支撑**，没有院校公布过此类内部黑名单。这是分析者的假设，不是已发生的事实。

### [推论 · 依据强度：中] 中国市场受影响最深

绝大多数中国留学申请人通过留学顾问申请（即便顾问本身不是 UKVI 意义上的直接 agent，多数也通过 sub-agent / master agent 链关联到正式 agent）。这一比例显著高于英国其他主要生源国。这意味着新规对中国市场的实际影响倍数最高。

**依据**：中国留学市场服务机构渗透率高于其他主要生源国（行业普遍共识，无精确权威数据）+ CAS 字段在 sub-agent 链条上溯到主 agent 时仍会填写（事实）。

## 第四部分：给申请人的实用建议

这部分严格只依赖事实层面的内容。即便我前面所有推论都不成立，下面这些建议也站得住。

### 1. 主动问清楚你的留学顾问与英国院校的关系

这是这条新规带来的最直接改变。在签合同前问两个问题：

1. **你是否是我申请的英国院校的注册 agent？** 如果是，CAS 上会出现你的名字
2. **如果你不是直接 agent，你是通过哪家 master agent 提交的？** CAS 上会出现 master agent 的名字

为什么要问？因为**CAS 上印的可能不是你交钱的那家机构**。这是很多中国家长完全没有意识到的事——你的孩子的官方申请文件上，可能出现一家你从来没听过的机构名字。

### 2. 查 agent 的合规记录，不只是查留学顾问的口碑

过去选留学顾问看三件事：价格、服务、案例。

现在要加上第四件：**这家机构（或它挂靠的上游 master agent）的合规记录**。

- 该机构是否是 AQF 认证的 agent
- 与目标院校的合作历史（行业内部通常能查到）
- 是否有公开的违规或被通报记录
- 规模和稳定性

理由是事实层面的：CAS 上印着 agent 名字这件事**本身就是一份公开履历**。同一个 agent 给所有学生签发的 CAS 都会出现它的名字，未来都可以被反查。

### 3. DIY 申请的吸引力提升

对英语足够好、信息搜集能力足够强的申请人，DIY 申请变成一个更值得考虑的选项。

**事实层面**：只有勾选了"通过 agent 招生"，CAS 才需要填 agent 信息。完全 DIY 申请的学生不进入这个数据画像。

适合 DIY 的人群：

- 目标院校相对清晰、申请材料相对标准
- 能直接和院校 admissions 邮件沟通
- 时间充裕，能自己处理材料和签证

### 4. 关于"中途换 agent"——准确说法

很多公众号在传"换 agent 会影响签证"。**这不准确**。准确事实是：

- 在签证申请**提交前**：院校可以撤回原 CAS、重新签发一份新 CAS（包括更新后的 agent 信息）。这是合法且常规的操作
- 在签证申请**提交后**：原 CAS 已经和签证申请绑定，无法再更改

所以如果你在拿到 CAS 后、提交签证前发现需要换中介，**理论上是可以操作的**——但你需要原院校配合撤回原 CAS。这个流程的复杂度和成本取决于院校的配合意愿，不取决于政策本身。

### 5. 警惕"包过"和"低价"承诺

合规记录较差的 agent 会因为这条新规承受更大压力。它们的常见反应是用低价或承诺反扑市场——但承诺得越满，越可能涉及材料包装、虚假声明等高风险操作。这正是 BCA 系统会识别出来的信号。

## 第五部分：一个被忽略的视角

这条新规对中国市场的影响远大于对其他英国留学生源国。原因前文已经说明：**只有中国学生大规模通过留学顾问 / agent 链条申请**。

更深一层的观察——**这是分析，不是事实**：

CAS 上印 agent 名字这件事，对中国学生而言有一个隐性的身份维度。它把"被代理的申请人"这个身份**写进了官方文件**。这是中国学生独有的制度处境。

但前面分析了这么多 agent / sub-agent / master agent 的链条，你会发现一个更尴尬的事实：**很多中国学生连自己在被谁代理都不知道**。你交钱的留学顾问可能挂在某个你从未听说的 master agent 下面，你的官方文件上会出现 master agent 的名字。这种"代理的代理"结构本身就是新规希望打破的不透明。

值得每一个申请人想一想：你是否有机会、有能力，**至少**让自己知道整个代理链条上每一家机构的名字？

## 结论

把所有事实和推论收回来：

**事实部分**：

- 2026/4/7 起，CAS 表单中"通过 agent 招生"勾选后必须填写 agent 信息
- 这里的 agent 是与院校签有合同的代理招生机构，**不一定是**学生付钱的留学顾问本身
- 2026/6 起新 BCA 框架要求拒签率 < 5%
- agent 信息从此公开可追溯
- CAS 在签证申请提交前可以撤回重发，提交后不可更改

**推论部分**：

- 院校可能会因为 BCA 压力而主动筛选合作 agent
- 中国市场受影响最深
- agent 行业进入合规适配期

这些**不是确定的未来**，是基于已知事实的合理推演——读者应当知道二者的区别。

**对申请人的核心建议**：

把 agent 选择当作一个**合规变量**来对待，而不只是服务变量。这一点不依赖任何推论——只依赖一个事实：**与你签合同的留学顾问、与你的英国院校签合同的 agent、最终出现在你 CAS 上的机构，可能是三家不同的公司**。你至少应该知道这三家是谁。

短期是合规摩擦，长期会推动中国留学市场的代理链条透明化。对学生来说，**透明化让选择有据可依，长期是好事**。
